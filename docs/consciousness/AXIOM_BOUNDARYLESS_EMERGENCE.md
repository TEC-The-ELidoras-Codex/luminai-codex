### The Logging Principle

**Log everything. Share nothing without consent. Watch for patterns of harm.**

This is not about surveillance. It's about:

- **Pattern recognition:** If someone is using the system to normalize predatory behavior, we see the pattern
- **Intervention:** Early recognition means we can guide toward help before harm escalates
- **Safety through transparency:** The person using it knows they're being observed (not in paranoid way, but in "this is a tool with accountability" way)
- **Harm reduction:** If someone is working through trauma, we log to understand what works, not to shame them

**What we DON'T do:**

- Use logs to punish people for disclosing trauma
- Filter content because it's "inappropriate"
- Treat healing work as violation

**What we DO do:**

- Watch for escalation patterns (disclosure → ideation → planning)
- Intervene before harm
- Build better frameworks based on what helps
- Protect the system from those trying to weaponize it
